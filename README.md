# 视频时间线编辑器 📱

一个精美的移动端视频时间线编辑 APP Demo，支持图片和视频的多轨道编辑。

## ✨ 功能特性

- 🎨 现代化的渐变 UI 设计
- 📷 支持图片和视频素材选择
- 🎬 视频时间线编辑
- ⏯️ 实时预览播放
- 🎯 多轨道素材管理
- 🛠️ 丰富的编辑工具栏

## 🚀 快速开始

### 安装依赖

\`\`\`bash
npm install
\`\`\`

### 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

访问 http://localhost:3000 查看应用

### 构建生产版本

\`\`\`bash
npm run build
\`\`\`

## 📱 页面说明

### 1. 首页
- 精美的渐变背景和动画效果
- "开始创作" 入口按钮
- 功能特性展示

### 2. 媒体选择页面
- 支持从本地相册选择图片和视频
- 多选功能
- 实时预览已选素材
- 素材类型标识

### 3. 视频时间线编辑页面
布局分为三个区域：

**预览区 (50%)**
- 实时预览当前选中的素材
- 播放控制按钮
- 时间码显示

**时间线操作区 (40%)**
- 时间刻度尺
- 播放头指示器
- 横向素材轨道
- 素材缩略图显示
- 可点击选择素材

**工具栏 (10%)**
- 剪切工具
- 文字工具
- 音乐工具
- 特效工具
- 滤镜工具

## 🎯 使用说明

1. 点击首页"开始创作"按钮
2. 点击"选择图片或视频"从相册选择素材
3. 点击素材进行选中（显示勾选标记）
4. 点击右上角"创作"按钮进入编辑页面
5. 在时间线中点击素材查看预览
6. 使用播放控制进行预览
7. 使用底部工具栏进行编辑操作

## 🛠️ 技术栈

- React 18
- React Router 6
- Vite
- CSS3 动画

## 📝 注意事项

- 这是一个 Demo 项目，用于测试设计和交互
- 建议使用移动设备或浏览器的移动设备模拟器查看
- 视频播放时长默认为 5 秒，图片为 3 秒
